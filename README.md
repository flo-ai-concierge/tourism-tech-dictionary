# Tourism Tech Dictionary

**Tech concepts translated for tourism & hospitality professionals learning to code.**

Built by [FLOHOM](https://flohom.com) — a vacation rental technology platform.

---

## What is this?

If you work in travel, tourism, or hospitality and want to learn software development, this is for you. Every technical concept is explained using analogies from the industry you already know.

No CS degree required. No jargon without explanation.

## How to use it

Browse the `concepts/` folder. Each file covers one technical topic with:

- **What it is** — plain English definition
- **Tourism translation** — the concept explained using hotel/travel analogies
- **When you'll encounter it** — real scenarios where this matters
- **Related concepts** — what to learn next

## Categories

| Category | What's covered |
|---|---|
| `infrastructure/` | Servers, memory, deployment, hosting |
| `development/` | Git, code, builds, packages |
| `apis/` | Endpoints, webhooks, integrations |
| `databases/` | Tables, queries, migrations |
| `ai/` | Prompts, models, tokens, context |
| `security/` | Auth, tokens, environment variables |
