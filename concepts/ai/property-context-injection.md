# Property Context Injection

## What it is
A specific form of context injection for property management. When a team member mentions a property (by name or number), the system automatically fetches that property's recent reservations, occupied night count, open tasks, and operational state — then injects it into the AI's context before processing the message.

## Tourism Translation
**"A hotel manager's dashboard that auto-opens the property file the moment someone says 'Room 14.'"**

You walk up to the operations desk and say "What's going on with Room 14?" Before you even finish the sentence, the manager's screen shows Room 14's recent guests, housekeeping status, pending repairs, and occupancy history. No clicking, no searching — the system anticipated the need.

## Why it matters
This is THE fix that made Slack FLO match Platform FLO's intelligence. When Travis asked "how close are we to pump out on Flo 14," FLO already had Soteria's reservation data pre-loaded — 4 occupied nights since April 5, next guest James Hunt on April 20. FLO only needed one search (for the pump-out date) instead of five separate tool calls.
