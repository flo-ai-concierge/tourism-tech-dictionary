# Context Synthesis

## What it is
When a system has no direct data from its primary source but reconstructs usable context from related or secondary sources. Instead of giving up, it looks for alternative information — like using admin notes when no guest messages exist — and synthesizes that into a format the AI can work with.

## Tourism Translation
**Like when a guest calls the front desk but the call isn't recorded in the system.** The front desk agent writes a handwritten note about what the guest said. When the next shift starts, they use that note to continue helping the guest — even though there's no official call log. The note isn't the original conversation, but it has enough context to keep things moving.

## When you'll encounter it
- When an AI system says "not enough context" but related data exists elsewhere
- When admin notes or manual logs need to feed into automated systems
- When building fallback paths so AI doesn't give up when primary data is missing
