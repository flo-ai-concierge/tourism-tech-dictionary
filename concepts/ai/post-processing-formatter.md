# Post-Processing Formatter

## What it is
Code that runs after an AI generates text, fixing formatting issues (spacing, emojis, structure) that the AI can't reliably get right on its own. Instead of asking the AI to format perfectly, you let it focus on content and clean up the presentation with deterministic code.

## Tourism translation
Like an editor who proofreads every guest newsletter before it goes out. The marketing team writes the content (the AI), but the editor (the post-processor) enforces the style guide — consistent headers, proper spacing, correct branding. The writer focuses on what to say; the editor ensures it looks right.

## When you'll encounter it
- AI-generated Slack reports that need consistent section spacing
- Chatbot responses that need brand-compliant formatting
- Any AI output that goes directly to customers or stakeholders where sloppy formatting would look unprofessional

## Related concepts
- Prompt engineering — trying to get the AI to format correctly (less reliable)
- Template rendering — using code templates instead of AI for structure
- Output validation — checking AI output against rules before delivery
