# Contract Amendment vs. Re-issue

## What it is
Two different ways to update a contract that has already been signed:

**Amendment (Addendum):** A separate short document that attaches to the original contract and says "everything above stays the same, except add/change this one thing." Both parties sign the amendment; the original contract stays untouched.

**Re-issue:** A completely new version of the full contract that incorporates all changes and replaces the original. Both parties sign the new version as if it were a fresh agreement.

## Which to use when

| Situation | Better approach |
|---|---|
| Small, clearly scoped addition (one clause) | Amendment |
| Multiple changes scattered through the document | Re-issue |
| Original was just signed and ink is still fresh | Re-issue (cleaner) |
| Contract is months old and fully operational | Amendment (less disruptive) |
| Need to preserve the original signed document legally | Amendment |

## Tourism translation
Think of it like a hotel booking modification. An **amendment** is like calling the front desk to add a late checkout to your existing reservation — the original booking stays, you just attach a note. A **re-issue** is like cancelling and rebooking with all the updated details in one clean confirmation. Same end result, but the paper trail looks different.

## Key gotcha
A re-issued contract where the contractor has already signed the original is technically a new agreement — the contractor should acknowledge the changes. For minor additions where trust is established, this is typically handled informally (email confirmation). For material changes, get a fresh signature.
