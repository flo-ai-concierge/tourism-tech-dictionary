# Turnover (Vacation Rental Operations)

## What it is
A property that has both a check-out AND check-in on the same day. The cleaning team must clean, restock, and prepare the unit between guests — often with a tight window (checkout at 10 AM, check-in at 3 PM = 5-hour turnaround).

## Tourism Translation
**A hotel room flip — one guest leaves in the morning, housekeeping cleans, next guest arrives in the afternoon.**

In hotels, this happens every day for every room. In vacation rentals, turnovers are the highest-pressure days because:
- The cleaning window is fixed and tight
- The property must be spotless (guests expect hotel-level cleanliness in a home setting)
- Supplies must be restocked (toiletries, linens, kitchen essentials)
- Smart locks need code changes
- Any maintenance issues found during cleaning must be resolved before the next guest

## Types on a daily schedule
- **Turnover** (red): Check-out + check-in same day — highest priority, time-sensitive
- **Check-out only** (yellow): Guest leaving, no one arriving — still needs cleaning but less urgent
- **Check-in only** (green): Guest arriving, property already clean from previous prep

## Why it matters for tech
Daily schedule automations need to clearly label turnovers so cleaning teams know which properties are time-critical. A check-out-only property can be cleaned anytime, but a turnover must be done within the window.

## Related concepts
- Daily schedule automation — automated morning reports for operations teams
- Smart lock code rotation — changing access codes between guests
