# API Media Support Gap

## What it is
When a software product can handle media (images, videos, files) in its own app interface, but its API (the programming interface other systems use to connect) doesn't include that media data. The app shows everything, but when other systems ask for the data through the API, the media is missing.

## Tourism Translation
**Like a front desk screen that shows a guest's passport scan, but the system can't forward it to housekeeping.** The front desk agent sees the photo on their computer, but when they try to share the guest file with another department's software, only the text fields transfer — the photo stays trapped on the original screen. The information exists, but the pipeline between systems drops the media.
