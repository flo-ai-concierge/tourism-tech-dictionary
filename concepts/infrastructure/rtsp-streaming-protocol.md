# RTSP (Real-Time Streaming Protocol)

## What it is
A standard network protocol that cameras and video devices use to send live video streams to other software. It's like a universal plug — any NVR (network video recorder), dashboard, or monitoring app that supports RTSP can connect to the camera and view its feed. Without RTSP, cameras are locked into their own proprietary app.

## Tourism Translation
**A universal key card system for security cameras.** Imagine if every hotel chain used a different key card format — you'd need a separate reader for each brand. RTSP is the universal standard that lets any front desk system read any camera's feed. When a camera doesn't support RTSP (like VicoHome), it's like having a proprietary key that only works with one brand's reader — you can't integrate it into your centralized security dashboard.
